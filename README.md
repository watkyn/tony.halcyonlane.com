# Tony's Terminal

A CLI-themed personal website.

## Setup

```bash
npm install
bundle install
```

## Commands

```bash
npm test        # Run tests with coverage
npm run build   # Build minified JS
bundle exec jekyll build    # Build site to _site/
bundle exec jekyll serve    # test the site at localhost:4000
```

## Structure

- `src/js/` - Source code
- `assets/js/` - Built output (don't edit)
- `test/` - Tests
